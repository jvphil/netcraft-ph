# 🔧 netcraft-ph

> A collection of proposed network topologies for various use cases — designed, simulated, and documented by [@jvphil](https://github.com/jvphil).

---

## 📖 About

**netcraft-ph** is a growing repository of network topology proposals covering real-world scenarios — from small office setups to enterprise-grade redundant infrastructures. Each topology is simulated using Cisco Packet Tracer and documented with full configuration guides, IP addressing tables, and cost-benefit analyses based on the Philippine market.

Whether you're a student, a network engineer, or just someone who loves building networks — feel free to explore, learn, and build on these designs!

## 📌 Why netcraft-ph?

Network design is both a science and a craft. This repo is a personal journey of learning, building, and sharing network designs that are:

- **Practical** — Real-world applicable
- **Cost-conscious** — Philippine market pricing
- **Well-documented** — No brain clouds left behind 🐟
